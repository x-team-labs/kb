WebLLM Basic Example

This is a simple example and can be greatly improved on. A web worker is great for a simple task like this, but you might like to use a service worker instead for cross page communication and avoiding loading the LLM over and over again.
This example includes some basic history, but this can be improved by using techniques like sliding context windows, summarization, and other techniques. Short is, you will eventually hit the context window limit and need to handle this.
